About Listify - Todo Manager

A simple and efficient todo manager built with Flask and Bootstrap

About This Project

Listify is a simple Todo Manager application designed to help users efficiently manage their daily tasks. Developed using Python's Flask framework, this app provides a user-friendly interface to create, edit, and view tasks. It also allows users to store task details and track their progress with minimal hassle.

With its clean and intuitive design, Listify helps you stay organized and productive, whether you're managing personal tasks or work-related activities.

Key Features:

  • Add, edit, and delete tasks
  • View tasks in a simple, organized list
  • Responsive layout for all devices
  • Built with Flask and Bootstrap for a smooth experience
